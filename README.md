# markxvi-workflows

GitHub Actions workflows for the self-hosted runner on MarkXVI.

The runner on this machine handles all automation jobs defined here.

## Workflows

| File | Trigger | Description |
|---|---|---|
| `audiogram-generate.yml` | Martedì 10:00 UTC | Genera audiogram dall'ultimo episodio del podcast |
| `audiogram-publish.yml` | Giovedì 10:00 UTC | Pubblica gli audiogram sui social |
